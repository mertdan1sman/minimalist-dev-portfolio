## 🛠 Teknik Detaylar

- **Saf CSS & HTML:** performans odaklı geliştirildi.
- **Responsive:** Mobil cihazlar ve tabletlerle tam uyumlu tasarım.

Bu proje, web arayüzü geliştirme konusundaki temel bilgileri ve modern tasarım prensiplerini sergilemek amacıyla hazırlanmıştır.
